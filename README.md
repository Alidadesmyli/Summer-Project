# XAI Stability in Medical Imaging: CNN vs Vision Transformer

**Research Question:** How does explanation stability under clinically plausible input perturbations (Gaussian noise, contrast shift, rotation) differ between DenseNet-121 and ViT-B/16 in chest X-ray pathology classification?

## Overview

This project compares the **robustness of post-hoc explanations** generated by two architectures:
- **DenseNet-121** (CNN) — explained via GradCAM / GradCAM++
- **ViT-B/16** (Vision Transformer) — explained via Attention Rollout

Stability is measured using **SSIM** and **top-k IoU** between explanations on original vs perturbed images.

Dataset: [CheXpert](https://stanfordmlgroup.github.io/competitions/chexpert/) (Stanford, requires registration)
